#  Firebase Credentials Manager UI

##  Overview

The **Firebase Credentials Manager UI** is an internal web interface built for **QA engineers** to securely **view and manage automation credentials** stored in **Firebase Realtime Database**.

It supports multiple environments (**QA** and **UAT**). 

⚠️ **Internal Use Only**
